## Getting Started

### Yeoman

```bash
npm install -g yo
```

### jsFile Engine Generator
To install generator-jsfile-engine from npm, run:

```bash
npm install generator-jsfile-engine
```

Finally, initiate the generator:

```bash
yo jsfile-engine engineName
